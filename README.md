## 🎯 About

> A curated collection of **Natural Language Processing** projects — from Twitter sentiment analysis to interactive NLP dashboards.

---

## 📂 Repository Structure

```
nlp-toolkit/
│
├── 📁 projects/
│   ├── twitter-sentiment-analysis/     # 🐦 Turkish tweet sentiment
│   ├── amazon-review-analysis/         # 📦 Product review NLP
│   ├── wikipedia-nlp/                  # 📚 Wikipedia text processing
│   ├── spam-email-detection/           # 📧 Spam classifier + Sentiment
│   └── nlp-interactive-dashboard/      # 📊 Full-stack NLP dashboard
│
├── 📁 fundamentals/
│   └── nlp_operations.py              # Core NLP utilities
│
└── 📁 docs/
```

## 🏆 Projects

| Project | Description | Tech |
|---------|-------------|------|
| 🐦 Twitter Sentiment | Turkish tweet sentiment analysis | NLTK, TF-IDF |
| 📦 Amazon Reviews | Product review classification | Scikit-learn |
| 📚 Wikipedia NLP | Large-scale text processing | NLP Pipeline |
| 📧 Spam Detection | Email spam classifier | ML + NLP |
| 📊 NLP Dashboard | Interactive analysis tool | FastAPI + Frontend |

```bash
git clone https://github.com/Emrecaglar05/nlp-toolkit.git
cd nlp-toolkit
pip install nltk scikit-learn pandas numpy matplotlib
```
